# Spring-Bean-load
Spring Bean装配（属性注入）
1. 基本属性注入
2. 数组注入 <list>
3. 集合注入 <list> <set> <map>
4. 构造函数装配 <constructor-arg index="" type="" value="" />
5. 继承装配  <bean id="" parent="" class="" >
6. 自动装配  <bean .. autowire=""> (byName, byType, autodetect, no, default, constructor)  <beans .... default-autowire="">
